A WooCommerce extra variation images  plugin/extension that adds ability for shop/store owners to add variation specific images in a group.

== Description ==

You can able to add any number of images to your variable product variations
The WooCommerce Extra Variation Images extension allows you to add extra gallery images per variation on variable products within WooCommerce.
By default WooCommerce will only swap the main variation image when you select a product variation, not the gallery images below it.
This extension allows visitors to your store to be able to swap different gallery images when they select a product variation. 


Note: Please make sure WC product page must have a default set of gallery images and not override default gallery page any where.

== Installation ==

1. Upload the entire 'woocommerce-extra-variation-images' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
